工具箱网站 - V1.0

项目概述

创建包含多个AI工具的工具箱网站，每个工具独立页面，通过首页卡片导航访问。

功能需求

1. 首页功能

• 网格布局展示工具卡片

• 每个卡片包含：图标、名称、简短描述

• 点击卡片跳转到对应工具页面

• 响应式设计（电脑3列，平板2列，手机1列）

2. 工具卡片要求

// 示例工具数据
{
  id: "face-generator",        // 工具ID
  name: "AI人像生成",         // 工具名称
  description: "使用AI生成独特的人像图片",  // 简短描述
  icon: "🎨",                  // 图标（emoji或图片）
  path: "tools/face-generator/index.html"  // 工具页面路径
}


3. 第一版包含工具

1. AI人像生成（已提供，路径：C:\Users\smith\Downloads\generate_face-main\generate_face-main\index.html）
2. AI起名助手（待开发）
3. AI营养分析师（待开发）

技术方案

• 纯静态HTML/CSS/JS网站

• 无后端，无数据库

• 工具页面独立HTML文件

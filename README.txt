
# AI工具箱网站

一个集中管理各种AI小工具的网站，通过卡片导航访问各个工具。

## 🎯 功能特点
- 卡片式工具展示
- 点击卡片跳转工具页面
- 响应式设计（电脑/平板/手机）
- 纯静态网站，无需后端

## 🚀 快速开始
1. 下载代码
2. 将你的工具HTML文件放到 `tools/对应工具名/` 目录
3. 在浏览器打开 `index.html`



## 🛠️ 现有工具
1. ✅ AI人像生成
2. 🔄 AI起名助手
3. 🔄 AI营养分析师

## 🔧 技术要求
- 现代浏览器（Chrome/Firefox/Edge）
- 支持HTML5/CSS3/ES6+

## 📱 适配设备
- 💻 电脑端：3列卡片
- 📱 平板：2列卡片
- 📱 手机：1列卡片
